HAPPINESS&Co
Proyecto web cultural desarrollado para el módulo de Lenguaje de Marcas, Base de Datos y Programación.

Descripción
HAPPINESS&Co es una agenda cultural pensada para organizar y mostrar eventos de ocio, música, teatro, cine y exposiciones. El proyecto incluye una web completa con varias páginas, una base de datos relacional, una parte de programación en Java y un feed RSS con los eventos destacados.

Características
Página de inicio con eventos destacados y recomendados.

Sección de próximos eventos y historial de eventos.

Páginas de sobre nosotros, fuentes y contacto.

Fichas de evento con información ampliada.

Base de datos con usuarios, eventos, galerías, imágenes y favoritos.

Aplicación Java con menú de gestión en consola.

Feed RSS con los eventos de la sección principal.

Estructura
index.html / index.css: página principal.

eventos.html / eventos.css: próximos eventos.

historial_eventos.html / historial_eventos.css: historial.

sobre_nosotros.html / sobre_nosotros.css: información corporativa.

fuentes.html / fuentes.css: referencias utilizadas.

contacto.html / contacto.css: formulario de contacto.

BaseDeDatosHappiness-Co.sql: estructura y datos de la base de datos.

rss.xml: feed RSS del proyecto.

Clases Java: Usuario.java, Evento.java, Galeria.java, Favorito.java, proyectoHappinessCo.java.

Instalación
Clonar el repositorio.

Abrir los archivos HTML en un navegador.

Importar BaseDeDatosHappiness-Co.sql en MySQL.

Compilar y ejecutar el proyecto Java desde un IDE o terminal.

Abrir rss.xml desde un lector RSS o navegador compatible.

Uso
Navegar por la web desde el menú superior.

Consultar próximos eventos, historial y fichas individuales.

Usar el formulario de contacto para enviar información.

Ejecutar la aplicación Java para gestionar usuarios, eventos, galerías y favoritos.

Consultar el feed RSS para seguir las novedades de eventos.

Base de datos
La base de datos incluye tablas para usuarios, eventos, galerías, imágenes y favoritos, con relaciones por claves foráneas y vistas para consultas frecuentes.

RSS
El feed RSS publica los eventos de la sección principal con título, enlace, descripción, fecha, lugar, categoría e imagen.
